# MatrixCV - 一个简洁的简历投递系统

### Requirement

- npm v3.5.2
- node v8.10.0

### Deployment

`$> git clone https://github.com/owtotwo/MatrixCV.git`

`$> cd MatrixCV`

`$> npm install`

`$> node app.js`

Then open `http://localhost:3000/`.
